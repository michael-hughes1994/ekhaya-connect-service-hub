## Overview

ekhaya-connect is a browser-based healthcare platform designed to unify access, coordination, and everyday service delivery. It brings together hospital services, home access, and tool access so users can reach care-related resources from a single web entry point.

The project is a good fit for organizations or communities that want a simple online front end for healthcare workflows. Its layout is intended to make essential services easier to navigate from both facility and home environments, while keeping connected tools and services grouped in one place.
